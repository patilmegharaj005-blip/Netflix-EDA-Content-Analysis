# 📊 Netflix Content Analysis: EDA & Business Insights

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover content trends, audience preferences, and business insights.

## 🎯 Objectives
- Analyze Movies vs TV Shows distribution
- Identify top countries producing content
- Understand content growth trends
- Explore genres and ratings
- Generate business insights

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Key Insights
- Netflix focuses heavily on Movies (~70%)
- USA dominates content production
- Content growth increased after 2015
- TV-MA and TV-14 are most common ratings
- International genres are highly popular

## 📈 Visualizations
Includes:
- Bar Charts
- Line Charts
- Trend Analysis
- Genre Distribution

## 📌 Conclusion
Netflix follows a scalable content strategy focusing on movies and global audience expansion.
